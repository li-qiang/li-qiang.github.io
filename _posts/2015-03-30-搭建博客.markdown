# 环境：Ruby
- - - 
<br/>
## 安装Jekyll

{% highlight bash %}
gem install jekyll # 安装Jekyll
{% endhighlight %}

## 生成新的项目

{% highlight bash %}
mkdir myBlog       # 新建一个文件夹
cd myBlog          # 进入新建好的文件夹中
jekyll new myBlog  # 生成Jekyll项目
{% endhighlight %} 
 
## 启动本地服务器

{% highlight bash %}
jekyll serve       #使用浏览器打开 http://localhost:4000 
{% endhighlight %}

[Jekyll 文档](http://jekyllcn.com/)
